# 基于Cloudflare Worker的丁真图API

基于Cloudflare Worker的丁真图API，支持简单的流量控制；图片集图片不多，但都是好图；需要图片集邮箱联系.

可以作为随机瑟图等其他类似工具的模版借鉴、参考。

API访问：

```
https://dingzhen-api.cinea2.workers.dev/get
限流每分钟15次请求，重申：需要图片集邮箱联系。
```
